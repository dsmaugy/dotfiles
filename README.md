# my dotfiles

managed by [chezmoi](https://github.com/twpayne/chezmoi)

## usage

### init

```bash
chezmoi init git@github.com:dsmaugy/dotfiles.git
```

### updating

```bash
# pull in updates
chezmoi update

# add new stuff
chezmoi add

# re-add edits
chezmoi re-add
```

## general requirements

- python/pyenv
- go
- nvim
